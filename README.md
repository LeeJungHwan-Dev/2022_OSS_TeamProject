## 🔎 프로그래 소개
#### 🤖 Slack Chat_Bot
> 1. 간단한 응답에 대한 처리가 가능합니다.
>> * ex) hello -> hi 와 같이 지정한 값에 따른 응답이 가능합니다.
> 2. 간단한 배수 연산이 가능합니다.
>> * ex) 4 -> 8 , 2 -> 4와 같이 지정한 값에 따른 연산이 가능합니다.


## ⚙️ 사용 방법
#### 1. 해당 repo를 clone후 다음고 같은 명령어를 입력한다.
~~~
npm install
~~~
#### ⚠️ 해당 프로그램은 기본적으로 node 환경이 구성되어 있어야 합니다.
